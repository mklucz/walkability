Introduction

What you're seeing here is a set of maps generated with drogi, a Python module I wrote. The maps are intended to visualize urban walkability and identify unwalkable areas.

Methodology

In short: OpenStreetMap data about walkways is turned into a graph and walked, randomly and repeatedly, with an A* algorithm. The resulting paths and the areas causing paths to deviate from a straight line are then colored in. You can read about the details here.
